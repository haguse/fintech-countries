## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en/download/)
- [yarn](https://yarnpkg.com/) or npm

### Installation

Clone the repo

```sh
git clone https://github.com/haguse/fintech-countries.git
```

<br/>

Install packages

`yarn` or `npm install`

<br/>

Start

`yarn start` or `npm start`
<br/>
